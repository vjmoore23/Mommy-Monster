Fun Learning

Mommy Monster is a fun interactive web application for children ages 3- 6.Our website uses child-friendly navigation, making it easy for children to learn how to use a mouse and keyboard on a website. Children can learn their alphabet, basic shapes(square, triangle, circle, rectangle), colors(Red, Orange, Yellow, Green, Blue, Purple), numbers(1-10), and basic JavaScript commands.
